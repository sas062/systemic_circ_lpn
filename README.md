# systemic_circ_lpn
MAE 290A Final Project 

Simplified model of adult circulation.


Heart being modeled as a single chamber with time-varying current source.


Open loop.

Ax = b 


A = B G B*


where B is the incidence matrix, and G is a diagonal matrix with $g_{ii} = \frac{1}{R_i}$


x: pressure vector


b: flow vector
